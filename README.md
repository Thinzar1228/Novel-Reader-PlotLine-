# 📖 PlotLine

A Wattpad-inspired web application developed using PHP and MySQL that enables users to publish, manage, and read stories through an interactive online platform.

---

## 🔍 Overview

PlotLine is a story-sharing platform designed for both writers and readers. The system allows authors to create and publish stories chapter by chapter, while readers can explore a collection of stories, save their favorites, and enjoy a seamless reading experience.

This project was developed as a web application to demonstrate full-stack web development concepts, including user authentication, database management, content management, and responsive user interface design.

---

## 🎯 Objectives

* ✍️ Provide a platform for writers to publish and manage their stories.
* 📚 Allow readers to discover and enjoy stories online.
* 🔒 Implement secure user authentication and authorization.
* 💻 Demonstrate the use of PHP and MySQL in a real-world web application.
* 📱 Create a responsive and user-friendly reading environment.
